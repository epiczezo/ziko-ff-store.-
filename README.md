# ZIKO FF STORE

This is a ready-to-use Free Fire settings store website. It includes:

- Product showcase
- Payment section (Vodafone Cash)
- About section
- 5-star customer reviews
- Floating WhatsApp contact button

Instructions:
1. Open `index.html` in a browser to view the website locally.
2. To host online, create a GitHub repository and upload this file.
3. Enable GitHub Pages in repository settings to get the live website link.
